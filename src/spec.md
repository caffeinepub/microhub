# Specification

## Summary
**Goal:** Build a landing page for Microhub, an influencer marketing agency in Mumbai that connects brands with micro and nano creators.

**Planned changes:**
- Create hero section with headline, subtext, and dual CTAs for brands and creators
- Build problem section highlighting ad campaign pain points and positioning Microhub as solution
- Add "What We Do" section explaining four core services (matchmaking, campaigns, brand exposure, reporting)
- Create "Why Microhub" section with four unique value propositions
- Build "How It Works" section with 4-step process flow
- Add creator recruitment section with compelling message and CTA
- Include testimonials section with placeholder cards
- Build contact form with five fields (Name, Brand Name, Budget Range, Contact Number, Email)
- Implement Instagram integration and WhatsApp contact button
- Create footer with Mumbai location and relevant links
- Apply premium minimal design with black, white, and electric blue or neon green accents
- Implement scroll animations for section transitions
- Add mobile-first responsive design across all sections
- Optimize SEO for "Influencer Marketing Agency in Mumbai"

**User-visible outcome:** A modern, responsive landing page where brand owners can learn about Microhub's services, understand the process, and submit lead forms. Creators can also discover opportunities through a dedicated section. The site features premium minimal design, smooth animations, and strong calls-to-action.
